# VPS AFF 网站管理员手册

## 1. 适用范围

本文档面向后台管理员，说明当前后台系统的实际操作方式。后台入口包括：

- 登录页：`/admin/login`
- 产品管理：`/admin/products`
- 公告管理：`/admin/announcement`
- 配置管理：`/admin/settings`

当前后台能力覆盖：

- 管理员登录
- 产品列表查询、分页、添加、编辑、删除
- 中英文公告 Markdown 编辑与预览
- 网站标题、Logo、社交链接配置
- 退出登录

## 2. 登录流程

### 2.1 登录入口

访问：

```text
http://你的域名/admin/login
```

首次部署后的默认账号：

- 用户名：`admin`
- 密码：`admin123`

### 2.2 登录步骤

1. 打开登录页。
2. 输入用户名和密码。
3. 点击“登录”按钮。
4. 登录成功后，系统会将 Token 保存到浏览器本地，并自动跳转到“产品管理”页面。

### 2.3 登录失败与限制

- 用户名或密码错误时，页面会提示“用户名或密码错误”。
- 同一 IP 15 分钟内连续失败超过 5 次，会触发限速，提示稍后再试。
- Token 过期、无效或被清除后，后台接口会返回 401，并自动跳回登录页。

## 3. 后台界面说明

登录后，后台整体分为两部分：

- 左侧菜单：产品管理、公告管理、配置管理
- 右上角：当前管理员欢迎信息与“退出登录”按钮

导航规则：

- `/admin` 会自动跳转到 `/admin/products`
- 未登录直接访问后台页面时，会被 `AuthGuard` 重定向到 `/admin/login`

## 4. 产品管理

产品管理页用于维护 VPS 产品数据，支持搜索、分页、添加、编辑和删除。

### 4.1 页面入口

- 菜单名：`产品管理`
- 路径：`/admin/products`

### 4.2 页面功能概览

页面包含：

- 顶部搜索框：按“服务商”或“产品名称”搜索
- “添加产品”按钮
- 产品表格
- 分页器

表格默认展示字段：

- ID
- 服务商
- 产品名称
- CPU
- 内存(GB)
- 硬盘(GB)
- 流量(GB)
- 带宽(Mbps)
- 价格
- 操作

### 4.3 搜索与分页

搜索操作：

1. 在搜索框输入服务商名或产品名称关键字。
2. 按回车或点击搜索图标。
3. 系统会自动回到第 1 页并重新加载列表。

分页操作：

- 默认每页 20 条
- 可切换页码
- 可调整每页数量

### 4.4 添加产品

点击“添加产品”后会弹出表单对话框。

必填字段：

- 服务商
- 产品名称
- 位置
- CPU(核)
- 内存(GB)
- 硬盘(GB)
- 月流量
- 带宽
- 价格/年
- 下单链接 (AFF)

可选字段：

- 测评链接
- 备注

特殊输入规则：

- 月流量支持 `GB` 或 `TB`
- 带宽支持 `Mbps` 或 `Gbps`
- 价格右侧货币代码必须为 3 位字母，例如 `USD`
- AFF 链接和测评链接会校验 URL 格式

添加步骤：

1. 点击“添加产品”。
2. 填写表单。
3. 点击弹窗底部“确定”。
4. 页面提示“添加成功”，列表自动刷新。

### 4.5 编辑产品

编辑操作入口在表格“操作”列。

编辑步骤：

1. 找到目标产品。
2. 点击“编辑”。
3. 在弹窗中修改需要变更的字段。
4. 点击“确定”保存。
5. 页面提示“更新成功”，列表自动刷新。

说明：

- 编辑弹窗会自动预填已有数据。
- 当前后台编辑时默认按数据库原始单位回填，即月流量按 GB、带宽按 Mbps 展示。

### 4.6 删除产品

删除操作为软删除，不会直接物理删除数据库记录。

删除步骤：

1. 找到目标产品。
2. 点击“删除”。
3. 在二次确认框中确认删除。
4. 页面提示“删除成功”，列表中该产品消失。

说明：

- 删除后的产品不会继续显示在后台默认列表里。
- 前台首页也不会再展示该产品。

### 4.7 常见提示

- “获取产品列表请求异常”：通常表示后端不可用或登录状态失效
- “操作失败”：一般为表单校验未通过、URL 格式错误或接口返回业务错误

## 5. 公告管理

公告管理页用于维护前台显示的中英文公告内容。

### 5.1 页面入口

- 菜单名：`公告管理`
- 路径：`/admin/announcement`

### 5.2 页面结构

页面使用两个标签页：

- 中文公告：对应配置键 `announcement_zh`
- 英文公告：对应配置键 `announcement_en`

每个标签页都包含：

- 左侧 Markdown 编辑区
- 右侧实时预览区
- “保存更改”按钮

### 5.3 编辑与保存步骤

1. 打开“公告管理”页面。
2. 选择中文或英文标签页。
3. 在左侧输入 Markdown 内容。
4. 右侧会实时预览渲染效果。
5. 点击“保存更改”。
6. 页面提示“保存成功”。

### 5.4 使用建议

- 标题可使用 `#`、`##`
- 列表可使用 `-` 或数字序号
- 链接可使用标准 Markdown 语法

如果页面提示“获取配置失败”或“保存请求异常”，优先检查后台登录状态和接口可用性。

## 6. 配置管理

配置管理页用于维护站点标题、Logo 地址和社交媒体链接。

### 6.1 页面入口

- 菜单名：`配置管理`
- 路径：`/admin/settings`

### 6.2 可维护内容

基础设置：

- 网站标题（中文）`site_title_zh`
- 网站标题（英文）`site_title_en`
- Logo URL `site_logo`

社交媒体链接：

- Telegram 链接 `link_telegram`
- YouTube 链接 `link_youtube`
- X (Twitter) 链接 `link_x`
- 博客/测评网站链接 `link_blog`

### 6.3 编辑步骤

1. 打开“配置管理”页面。
2. 修改基础设置或社交媒体链接。
3. 点击底部“保存修改”按钮。
4. 页面会并行提交所有可编辑配置项。
5. 全部成功后提示“配置保存成功”。

### 6.4 输入规则

- 网站标题中英文为必填项
- `site_logo` 使用 URL 校验，当前为 warning 级提示
- 社交链接使用 URL 校验，不符合格式时无法正常提交

### 6.5 生效说明

- 保存后，前台相关内容通过 `/api/config` 获取
- Header 中的站点标题、Logo 和社交链接会读取这些配置

## 7. 退出登录

退出步骤：

1. 点击右上角“退出登录”按钮。
2. 系统会删除浏览器本地保存的 Token。
3. 页面跳转回 `/admin/login`。

如果需要重新进入后台，必须再次登录。

## 8. 操作建议与限制

建议：

- 首次上线后先完成公告和站点配置，再录入产品
- 添加产品时先确认 AFF 链接与测评链接有效
- 重要公告修改前，可先在本地文档中保存一份 Markdown 备份

当前后台限制：

- 后台暂未提供管理员密码修改页面
- 后台暂未提供已删除产品回收站页面
- 配置项更新基于现有内置键，不支持在后台新增自定义配置键
